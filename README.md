# WeChatClient
一个高仿微信网页版 https://wx.qq.com 的WPF应用程序，基本完成了网页微信的主要基础功能，例如聊天，联系人，搜索，表情等。

软件中的图标和表情都是使用网页微信的资源，还有很多功能是待实现的，例如任务栏托盘图标，图片文件发送，截屏，创建群聊等。

这里暂时提供一些软件截图。
